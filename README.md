# 💊 دكتور بوت - Medication Reminder Bot

> بوت تيليغرام ذكي لتذكير المرضى بأدويتهم بالوقت الصحيح مع لوحة تحكم على HuggingFace.

---

## ✨ المميزات

| الميزة | الوصف |
|--------|-------|
| ⏰ تذكيرات تلقائية | يذكر المريض بكل دواء بالوقت المحدد |
| ✅ تأكيد الأخذ | المريض يضغط زر "أخذته" لتسجيل الجرعة |
| ⚠️ تحذير الصلاحية | تنبيه يومي إذا دواء قرب على الانتهاء |
| 😟 تنبيه الجرعة الفايتة | تنبيه مسائي بالأدوية اللي ما أخذها |
| 📅 ملخص أسبوعي | تقرير أسبوعي بنسبة الانتظام |
| 🤖 دكتور بوت AI | مساعد ذكي بالعربي العراقي |
| 🖥️ لوحة تحكم | Dashboard لإدارة الأدوية بسهولة |

## 🚀 طريقة النشر على HuggingFace

### الخطوة 1: إنشاء البوت على تيليغرام

1. افتح تيليغرام وابحث عن `@BotFather`
2. اكتب `/newbot`
3. اختر اسماً للبوت
4. احفظ **Bot Token** اللي يعطيك إياه

### الخطوة 2: إنشاء Space على HuggingFace

1. روح على [huggingface.co/spaces](https://huggingface.co/spaces)
2. اضغط **New Space**
3. اختر:
   - **SDK:** Gradio
   - **Visibility:** Public أو Private
4. ارفع الملفات:
   - `app.py`
   - `patients.json`
   - `taken_log.json`
   - `requirements.txt`

### الخطوة 3: إضافة الأسرار (Secrets)

في صفحة الـ Space:
1. اضغط **Settings**
2. روح لـ **Repository Secrets**
3. أضف:

| الاسم | القيمة |
|-------|--------|
| `BOT_TOKEN` | توكن البوت من BotFather |
| `GROQ_API_KEY` | مفتاح Groq من console.groq.com (اختياري) |
| `ADMIN_PASSWORD` | كلمة سر لوحة التحكم |

### الخطوة 4: تشغيل البوت

بعد رفع الملفات، الـ Space يشتغل تلقائياً!
- البوت يبدأ polling على تيليغرام
- لوحة التحكم تظهر على رابط الـ Space

## 📱 طريقة الاستخدام

### للمريض (تيليغرام)

```
/start    - ابدأ مع البوت
/menu     - القائمة الرئيسية
/mymeds   - عرض أدويتي
/taken    - سجل أنك أخذت دواء
/reset    - مسح المحادثة
```

### للمشرف (Dashboard)

1. افتح رابط الـ HuggingFace Space
2. من التبويبات:
   - **تسجيل مريض**: أضف مريض جديد بـ Chat ID
   - **إضافة دواء**: أضف دواء مع الأوقات والصلاحية
   - **عرض الأدوية**: شوف أدوية مريض معين
   - **حذف دواء**: احذف دواء من القائمة

### كيف تعرف Chat ID؟

1. افتح تيليغرام
2. ابحث عن `@userinfobot`
3. اكتب `/start`
4. سيعطيك Chat ID الخاص بك
