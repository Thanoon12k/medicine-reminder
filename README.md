# 💊 دكتور بوت v3 — AI Medication Reminder Bot

بوت تيليغرام ذكي لتذكير مقدمي الرعاية بمواعيد أدوية مرضاهم.
**كل شي يتم من تيليغرام مباشرة — بدون داشبورد إضافي.**

---

## ✨ المميزات الجديدة في v3

| الميزة | الوصف |
|--------|-------|
| 🤖 إدخال ذكي | اكتب الأدوية بأي أسلوب والذكاء الاصطناعي يفهمها |
| 👥 متعدد المرضى | سجّل أبوك وأمك وجدك كلهم في بوت واحد |
| 💬 تسجيل تلقائي | كل شي من تيليغرام، بدون داشبورد |
| ✅ تأكيد الجرعة | زر "تم إعطاء الدواء" مع كل تذكير |
| ⏰ Snooze | "ذكرني بعد 15 دقيقة" |
| ⚠️ تحذير الصلاحية | تنبيه يومي للأدوية قاربت تنتهي |
| 😟 جرعات فايتة | تنبيه مسائي بكل الجرعات اللي ما أُعطيت |
| 📅 ملخص أسبوعي | تقرير كل أحد بنسبة الانتظام |
| ✏️ تعديل وحذف | عدّل أي دواء أو احذف مريض من البوت مباشرة |

## 🚀 طريقة النشر على HuggingFace

### 1. أنشئ البوت من @BotFather
```
ابحث عن @BotFather في تيليغرام
اكتب /newbot
احفظ التوكن
```

### 2. أنشئ Space على HuggingFace
- روح [huggingface.co/spaces](https://huggingface.co/spaces)
- New Space → SDK: Gradio
- ارفع الملفات الأربعة:
  - `app.py`
  - `users.json`
  - `taken.json`
  - `requirements.txt`

### 3. أضف الأسرار في Settings → Secrets

| الاسم | المصدر |
|-------|--------|
| `BOT_TOKEN` | @BotFather |
| `GROQ_API_KEY` | [console.groq.com](https://console.groq.com) |

### 4. شغّل!
بعد رفع الملفات، الـ Space يبدأ تلقائياً.

## 📱 كيف يستخدمه المريض/مقدم الرعاية

### الخطوة 1 — ابدأ البوت
```
/start
```
ستظهر رسالة ترحيب مع زر "إضافة مريض".

### الخطوة 2 — أضف مريض
اضغط **➕ إضافة مريض** واكتب الاسم.

### الخطوة 3 — اكتب الأدوية بحرية
```
أموكسيسيلين حبة كل 8 ساعات بعد الأكل
ميتفورمين 500 مرتين يومياً الصبح والمساء
فيتامين د حبة واحدة الصبح، صلاحيته 2025/12/31
```
الذكاء الاصطناعي يستخرج كل شي تلقائياً! 🤖

### الخطوة 4 — استلم التذكيرات
```
🔔 وقت الدواء!
👤 المريض: أبو علي
💊 الدواء: أموكسيسيلين
📋 الجرعة: حبة واحدة
🕐 08:00

[✅ تم إعطاء الدواء]  [⏰ بعد 15 دقيقة]
```
